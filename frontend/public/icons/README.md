# Icon cho nút Đăng ký & Tải xuống (trang chủ)

Các logo icon kiểu “sách vàng, khiên đỏ trong khung kim cương” trong mẫu thường **không phải vẽ bằng CSS hay Canvas** mà là **ảnh tĩnh** do designer làm (Photoshop, Illustrator, Figma) rồi export PNG/WebP nền trong suốt.

## Cách dùng

Đặt ảnh vào thư mục này với tên:

- **icon-registration.png** (hoặc .webp) — icon cho nút Đăng ký (sách/cuộn vàng, khung kim cương).
- **icon-download.png** (hoặc .webp) — icon cho nút Tải xuống (khiên đỏ–vàng, khung kim cương).

Nếu không có file, trang sẽ tự dùng icon SVG đơn giản (fallback).

## Gợi ý làm icon

1. Nhờ designer vẽ và export PNG/WebP nền trong suốt (khoảng 128–256px).
2. Hoặc tìm icon pack game/fantasy có sẵn (có bản quyền).
3. Hoặc dùng SVG export từ Illustrator/Figma nếu muốn scale sắc nét (cần đổi code dùng `<img src="/icons/xxx.svg">` hoặc import SVG).
